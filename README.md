# Basic Breakdown

Basic Breakdown is a mobile-first newsletter website that turns complex news into short, clear, digestible explanations. Posts include structured sections, summary bullets, and optionally AI-generated infographics created via an n8n workflow using Nano Banana AI.

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).
